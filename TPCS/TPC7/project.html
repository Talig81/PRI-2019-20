Abstract

Neste pequeno projeto, iremos proceder a criação de um servidor em NODEjs utilizando a framework Express e como base de dados o MongoDB. Este pequeno servidor tem como objetivo fornecer uma pequena API de dados para que possamos fazer pedidos ao servidor e ele nos responder com a informação pretendida.

O dataset usado será o obras.xml que depois é importado para a base de dados mongo para preencher a coleção de obras. Após este procedimento, iremos proceder a definição das rotas e dos controladores. As rotas servem para responder a pedidos feitos ao servidor que depois chamam o controlador e ele devolve a informação que nos pedimos. Para além disso, é preciso definir um modelo que terá como referencia a constituição de uma obra na base de dados Mongo.
