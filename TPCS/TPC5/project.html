Abstract

O Express.js, é uma framework para node que nos permite criar aplicações de forma rápida, fléxivel e eficaz. A partir do express, temos acesso a várias bibliotecas e funcionalidades que tornam todo o processo de uma criação de uma aplicação muito minimalista.

Os mecanismos fornecidos por esta framework consistem em definir configurações comuns numa aplicação web como a porta a ser usada, a localização dos modelos, as renderizações através de ficheiros na pasta "views", etc... Apesar de o express parecer complexo, a sua boa documentação e a sua resolução de problemas, permitem aos developers uma rápida maneira de desenvolver aplicações web.

Estes projecto, consiste na criação de um Gestor de Alunos ao qual permite listar todos os alunos com a sua respetiva lista de notas. Para além de listar, também deve permitir ao utilizador, adicionar alunos, adicionar notas e assim como eliminar alunos e eliminar notas através do seu indicador. Para iniciar o servidor, basta correr o comando "npm start" após as dependencias terem sido todas
