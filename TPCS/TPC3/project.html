Abstract

O XSL é uma linguagem de marcação em XML utilizada para criar ficheiros XSL que definem como os ficheiros XML serão apresentados pelo browser ou aplicativos que o suportam. Esta linguagem, utilizada o XPATH como forma de acessar ou referir partes de um documento XML.

A partir de um documento de entrada e utilizando as definições de um ficheiro XSLT, é possivel mudar a representação nesse ficheiro utilizando outros estilos como por exemplo HTML. Estas transformações, consistem na travessia de todos os nodos listados e a cada nodo percorrido, é aplicado uma regra. Depois de formar todas as regras, é gerado o ficheiro com a representação definida.

Neste Project Record, é necessário a criação de um ficheiro HTML com o nome Project Record, gerado a partir de uma stylesheet XSLT e tem como documento de entrada, o ficheiro em XML gerado pelo template DTD desenvolvido na aula. A seguir, utilizando o mesmo processo anterior, é realizado o exercicio da geração de um website que contenha todos os arqueossitios fornecidos por um ficheiro XML.
