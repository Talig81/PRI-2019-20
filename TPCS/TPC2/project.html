Abstract

Este project record, consiste na realização de um documento DTD (Document Type Definition) que tem como objetivo a definição de uma estrutura XML para que seja possível, fazer uma validação de documentos de este tipo de maneira mais eficaz e mais fácil. Para além do documento, também é necessário gerar o ficheiro XML e preenchê-lo com os devidos parametros. Depois, será necessário responder a queries de diferentes ficheiros de XML utilizando o XPATH. Esta linguagem de rotas, permite-nos navegar muito facilmente através dos nodos XML. Por fim, este project record será apresentado numa página HTML com a estrutura semelhante ao ficheiro XML gerado assim como a informação toda sobre o desenvolvimento deste project record.
