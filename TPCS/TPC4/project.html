Abstract

O Node.js é uma plataforma que permite a construção de aplicações WEB de maneira escalável e de baixo consumo de recursos. Através da sua interpretação de JavaScript é possível a criação de servidores que conseguem responder a milhares de pedidos/eventos em tempo real utilizando uma unica maquinaa fisica.

O objetivo deste projecto, consiste na criação de um servidor em Node.js que recebe pedidos para abertura de pequenos documentos XML e responde com a transformação de esses ficheiros em HTML utilizando uma stylesheet XSLT denominada "doc2html.xsl". Como ponto de entrada, temos um grande ficheiro em XML que contêm toda a informação para este pequeno projecto. A partir desse ficheiro e utilizando outra stylesheet em XSL dividimos esse documento em pequenos ficheiros "doc.xml". Depois, a cada caminho disponibilizado pelo utilizador, o servidor irá responder com o ficheiro correspondente a esse caminho.

Neste Project Record, ao abrirmos o servidor, basta simplesmente introduzir o número do documento pretendido depois do caracter especial "/" e o servidor através do ficheiro "doc2html.xsl" envia o documento em xml pretendido e após a transformação utilizando a stylesheet, ele irá enviar ao utilizador a página html correspondente.
